The Challenge

GERIC faced difficulties tracking document statuses, a lack of centralized storage, and time-consuming manual reporting processes. I was tasked to replace a legacy XAMPP setup with a modern, secure architecture.

Technical Execution

Framework and Backend

Developed using PHP and the Laravel framework with a modular architecture for high maintainability.

Database Design

Engineered a MySQL database including an Entity Relationship Diagram (ERD) to manage complex data flow across tables.

System Architecture

Implemented a three-layer model (User Interface, Domain, and Data Access) to improve security and scalability.

Security

Integrated Role-Based Access Control (RBAC) to restrict functions and maintain data integrity based on user permissions.

Process Modeling

Created Level-0 Context Diagrams and Use Case Diagrams to define system interactions for Admin and Non-Admin actors.

PHP Laravel MySQL 8.0 RBAC MVC Architecture ERD DFD Use Case Diagrams
